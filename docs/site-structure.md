## Purpose

UBI_James set up a series of community forum threads in the TDResurgence Discord to gather structured player feedback. @imajes has consolidated the raw discussions into reviewable summaries. This site presents those summaries in a format useful to the development team.

Trello board (TDR team): https://trello.com/b/qTP1a9cv

## Page Templates

### Homepage
1. Hero ÔÇö two-line context statement + two headline stats (total threads, total participants)
2. Bucket grid ÔÇö eight cards: icon, name, thread count, one-liner
3. Top Priority Asks strip ÔÇö 3ÔÇô5 highest-consensus asks pulled across all threads
4. Footer ÔÇö credits, Trello link, date of last update

### Bucket Landing
1. Bucket title + one-paragraph framing
2. Thread cards ÔÇö one per thread in bucket: title, ask type badge, participant count, sentiment badge, one-line summary
3. Common themes callout ÔÇö recurring asks that span multiple threads in the bucket

### Thread Page
1. Header bar ÔÇö title, bucket label, ask type badge, date range, participant count
2. Metrics row ÔÇö message count, unique participants, reaction total, sentiment badge
3. Summary paragraph
4. Primary Asks ÔÇö each as a card: the ask, who raised it, who echoed it, category badge
5. Community Voices ÔÇö 2ÔÇô3 key quotes, Discord-style
6. Assessment ÔÇö prose, editorial treatment
7. Related threads ÔÇö same bucket + overlapping asks
