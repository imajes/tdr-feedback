Uppercase bucket tag with a colour-coded swatch. Four known buckets: specs (violet), balance (gold), features (blue), bugs (red).

```jsx
<CategoryBadge category="balance" />
<CategoryBadge category="bugs" />
<CategoryBadge color="var(--color-accent)">Live Ops</CategoryBadge>
```

- The swatch is the only colour; the chip body stays neutral so badges read as a calm set, not a rainbow.
